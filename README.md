# DaylightSaved

Daylight (savings time)? Saved.

Probably far from the first person to come up with the idea, but just puts calendar events (via .ics subscription/import) for reference to any Daylight Savings timezone changes.

Motivation: I work on CME related trading from Sydney. Who knows when Chicago starts to do DST? Or for that matter, how does it even line up with Sydney which also does DST?! What system settings (hopefully none!! because just describe everything with TZ!!!) need to change??
